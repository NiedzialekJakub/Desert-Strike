Desert Strike is a third-person shooted game built in Unreal Engine 5.

About the Project
This is a gameplay-focused project showcasing key gameplay mechanics developed from scratch, including:
	- Multiple character classes, such as the main player and AI-controlled enemies.
	- Enemy AI that can follow the player, aim, and shoot.
	- UI widgets, including a HUD that tracks:
		Number of enemies killed
		Amount of gold collected
	- Game ending condition: the game ends when the player collects all the gold available on the map.
	- The game is set in a desert enviroment, featuring varied terrain and cover elements.

All assets used in the project were sourced from free rescources available on Fab.
